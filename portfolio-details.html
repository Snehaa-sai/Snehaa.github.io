Aadhar Card online web application




Tech Stack

  • ToolKit: Eclipse, Visual Studio Code
  • Frontend: HTML, CSS, Angular, Typescript
  • Backend: JAVA
  • Database: MySQL

"A Web Application to automate the process of applying for an Aadhar Card by making it smoother for Indian citizens."

It consists of Two section:

Admin Portal:

In this application admin can do certain operation like,

  • Login through admin credentials
  • Approve new Aadhaar Card request
  • Verify request for duplicate Aadhaar
  • Display all issued Aadhaar Card
  • Delete Aadhaar card details for dead citizen

User Portal:

In this application user can do certain operation like,

  • Sign in to apply for a new Aadhar Card
  • Login to see the Aadhar number assigned by the admin
  • Update address, phone number, and date of birth of Aadhaar Card
  • Request duplicate Aadhaar Card
