Ticket Booking Web Application




Tech Stack

  • ToolKit: Eclipse, Visual Studio Code
  • Frontend: HTML, CSS, Angular, Typescript
  • Backend: JAVA
  • Database: MySQL

"It is dynamic and responsive web application for booking movie tickets online for different genres and languages."

It consists of Two section:

Admin Portal:

In this application admin can do certain operation like,

  • The admin can create account and able to login into account.
  • It can able to add or remove diiferent genres to or from application to build a rich prroduct line.
  • Admin can able to do edit movie details like name, ticket price, language, description and show timing to keep it aligned to the current price.
  • Once admin is authenticated, the admin can be used to enable or disbale movie show.
  • the admin can delete the movies and able to view different genres.

User Portal:

In this application user can do certain operation like,

  • The user can able to register to the application to book tickets.
  • After registering an account, the user able to update their accounts and able to change the password.
  • Sign into the application to maintain the record of activities.
  • The user can able to search for movie tickets based on the search keyword.
  • Apply filters and sort results based on different genres.
  • User can add all the selected movie ticket to a cart and customize the purchase at the end.
  • Experience a seamless payment process.
  • Receive a booking summary page once the payment is complete.
